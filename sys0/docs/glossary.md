# Glossary

*A reference for abbreviations and shorthand used across the repo.*

---

## Folder Structure

| Term | Full form | Contains |
| --- | --- | --- |
| `sys0` | System 0 | Repo-level stuff docs, core pages like `sitemap.html` |
| `sys1` | System 1 | All learning projects and shared resources |
| `lab` | Laboratory | Prefix for all learning platform folders e.g. `lab-odin`, `lab-codecamp` |
| `p1`, `p2`, `pX` | Project 1, Project 2, Project X | Individual project folders inside each lab |

---

## Git & Versioning

| Term | Full form | Notes |
| --- | --- | --- |
| `dev` | Development | The integration branch where all work flows through before reaching `main` |
| `v` | Version | Prefix for all version tags e.g. `v1.4.1` |
| `X.Y.Z` | MAJOR.MINOR.PATCH | The three parts of a version number. See [conventions.md](./conventions.md) for when each is bumped. |
